Hick's Law

Hershey's
hershey's.com

This design makes use of Hick's Law because the menu options are minimized to be easy for users to choose from. It also helps users in taking the following action smoothly, given the small number of options it presents.

PARC: Repetition

Home Depot
homedepot.com

Repetition using orange color on this page binds together the page components. Buttons, symbols, and even guidelines are reliably orange and differentiation with the text and background. There are numerous components on this page, and using a constant color makes amicability by making the page not show up so occupied and jumbled. It permits the data to be given, yet in a fair way that doesn't divert from the items.

PARC: Alignment

Walmart
walmart.com

We can see a beautiful lineup on the website. It repeats the same pattern of alignments and helps make the layout clear and readable. Because grids neatly organize product text and images, they create unity in the design. It attracts the attention of the users and helps to increase the desire for the product.
